StoryTime Discord Bot

StoryTime takes in a word(s) prompt and returns a custom AI generated story starting with those words.

To Install

  • Make sure you are logged in to Discord on web
  • You must have a role with Administrator or Manage Server permissions enabled in the server you want to add StoryTime to
  • Click here and choose the server to add StoryTime

Usage Rates

Due to server costs, unfortunately there has to be a limit on calls per minute per user.

Basic 3 requests per minute
